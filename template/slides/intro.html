Intro to My Slideshow

  • Each slide is its own HTML file in /slides.
  • Order lives in deck.json (insert slides without renaming).
  • After slide order is adjusted in deck.json, run build_deck.py to udpate the deck.
  • Navigation is injected automatically (Prev / Index / Next). index.html and print.html updated automatically.
  • Print everything via /print.html. Most interactivity is intentionally disabled in the print version.
